## 문서 사용 가이드

1. 프로젝트 전체 사용법은 루트 `README.md`를 먼저 확인합니다.
2. 데이터셋 전처리/경로/평가 기준은 `data/` 문서를 참고합니다.
3. 지표 해석 및 계산 기준은 `metrics.md`를 참고합니다.
4. 리더보드 결과를 추가하거나 검토할 때는 `leaderboard_update_workflow.md`와 `result_submission.md`를 함께 확인합니다.
